## Description
A dynamic music player interface developed for **Task 4** of the **CodeAlpha Internship**. It overrides default browser audio controls to provide a custom, stylish listening experience.

## Features
- **Playback Controls:** Play, Pause, Next, and Previous functionality.
- **Progress Bar:** Visualizes song duration and allows clicking to seek specific times.
- **Dynamic UI:** Album art rotates when music is playing and updates based on the current song.
- **Song Info:** Displays the current song title.

## Technologies Used
- HTML5
- CSS3 (Animations & Gradients)
- JavaScript (Audio API)

## How to Run
1. Clone the repository.
2. Open `index.html` in a web browser.
3. Click the "Play" button to stream the music.
